## Packages/Requirements
- Python 3.X.X
- Graphviz 0.13.2
## Nodes and Edges
### Defining a Node
```
c.node(‘node id string’, ‘node label’)
```
### Defining an Edge
```
c.edges([(‘node id string1’, ‘node id string2’), (‘node id string2’, ‘node id string3’)]
```
